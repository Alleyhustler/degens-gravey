About...

Cloak of Darkness is a porting project, started by Roger Firth, meant to help future creators explore different IF systems based on one same game. It is, if you will, the IF-version of the "Hello, world" test.

A Cloak of Darkness port is a short game, set in an empty Opera House, where the goal is to find and read a left-behind message. The player wears a dark cloak, which must be interacted with to reach the end scene.

Though there are already many known available ports, I still wanted to make my own for fun. First with the formats I was already familiar with, then expanding to programs I have yet to learn. I managed to contribute to that project with a couple of my ports.

While each port includes the use of basic commands of each system, none will claim to be the most correct way of doing those specific mechanics. The download points include a playable file and source code for the indicated engine - making importing to their respective programs easy.
