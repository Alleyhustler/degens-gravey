About...

The 100% Good Twine SugarCube Templates is a series of templates, made for the SugarCube format of Twine, available for free to any creator wanting to use pre-made interfaces or include accessibility elements in their games. This is an ongoing project, with new templates released when inspiration strikes me. They are often based or inspired by one of my projects.

Each template is ready-to-use, easy-to-edit, annotated, and always freely available on itch.io. They all include explanations for code written in JavaScript, HTML, CSS, and TwineScript. Users can choose between two download points: the full Zip folder with the source code and assets, or the basic HTML file.

Templates Available:

User Interfaces:

  • Simple Book Temaplate (landscape-orientation)
  • One Page Template (portrait-orientation)
  • Space-Tech UI Template (portrait-orientation)
  • VN-lite RPG Template (portrait-orientation)
  • Title Page Template (all use)

Other codes:

  • Settings Template (most requested/used settings)
  • Character Creator Template (common PC traits and interactivity)

Warnings

Before using the templates, it is recommended you change the IFID of the file. This is partly for archival reasons, but also to avoid issues when importing projects.
